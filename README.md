# SPALF: Spectrally-Preconditioned Augmented Lagrangian on Stratified Feature Manifolds

Standard sparse autoencoders (SAEs) minimize a weighted sum of reconstruction error and sparsity. This requires manually tuning the weight between the two objectives, and the right setting depends on both the data and the optimizer. SPALF removes this tuning by treating SAE training as constrained optimization: minimize $\ell_0$ sparsity subject to faithfulness, lexical drift, and orthogonality constraints, solved via an augmented Lagrangian with adaptive dual updates.

**Stratified decoder.** The decoder has two blocks. An *anchored* block $W_A \in \mathbb{R}^{d \times V}$ is initialized from the model's unembedding matrix and kept aligned with vocabulary directions through a drift constraint, giving each anchored feature a direct lexical interpretation. A *free* block $W_B \in \mathbb{R}^{d \times (F-V)}$ is learned from scratch under unit-norm projection to capture non-lexical structure. The anchored encoder rows are set to whitened vocabulary vectors $W_{\text{enc}}^{[:V]} = (\Sigma^{-1/2} W_{\text{vocab}})^\top$ so that each pre-activation measures similarity in the whitened metric. The free encoder rows are QR-orthogonalized against the anchored subspace, and the decoder bias is set to the data mean.

**Smooth JumpReLU gating.** Features are gated by JumpReLU with learnable per-feature log-thresholds. The step function in JumpReLU is non-differentiable, so during backpropagation it is replaced by a $C^1$ Moreau envelope surrogate with per-feature bandwidth $\gamma_j$ set from the interquartile range of pre-activations near each threshold. Unlike the rectangle kernel used in ([arXiv:2407.14435](https://arxiv.org/abs/2407.14435)), this surrogate has no gradient discontinuity at the decision boundary. Both the $\ell_0$ objective and constraint violations receive gradients through this surrogate via fused Triton kernels.

**Constrained augmented Lagrangian.** The augmented Lagrangian uses a smooth penalty $\Psi(g, y) = (\max(0, 2g+y)^2 - y^2)/4$ ([arXiv:2510.20995](https://arxiv.org/abs/2510.20995)) that transitions smoothly between active and inactive constraints. Dual variables are updated with a proportional-integral (PI) rule whose proportional gain matches the penalty coefficient, which makes the dual update equivalent to gradient descent-ascent on the augmented Lagrangian ([arXiv:2509.22500](https://arxiv.org/abs/2509.22500)). Each constraint has its own penalty weight that only increases over time and adapts to how noisy each constraint signal is. Constraint violations are smoothed with a bias-corrected EMA, and dual updates are deferred while the primal is still making progress within tolerance.

**Residual-coupled annealing.** A single scalar $r(t) \in [0, 1]$ measures how far training is from satisfying the constraints and drives three coupled schedules: (1) the surrogate bandwidth shrinks as $\gamma = \max(r, \alpha_{\text{floor}}) \cdot \gamma_{\text{init}}$, making the surrogate approach the true step function as constraints are met, with $\alpha_{\text{floor}} = (d/F)^2$ preventing excessive gradient amplification; (2) a discretization coefficient $\lambda_{\text{disc}} = \max(1 - r, \lambda_{\text{disc}}^{\text{prev}})$ grows monotonically to close the gap between the surrogate $\ell_0$ and the true $\ell_0$; (3) once all reconstruction constraints are satisfied, a KL-divergence constraint activates, shifting the fidelity target from whitened reconstruction error to the distributional match between original and SAE-patched next-token predictions.
